Windows 버튼은 최신 릴리스의 파일을 받습니다.

✨ 주요 기능

  • 펼치기 / 접기 — 화면 끝 색 탭에 마우스를 올리면 펼쳐지고, 빼면 접힙니다.
  • 작성 / 편집 — 펼친 메모 본문을 클릭해 마크다운으로 쓰고, 다른 곳을 클릭하면 렌더된 보기로 바뀝니다.
  • 위치 이동 — 상단 손잡이(⋮⋮)를 드래그하면 가까운 화면 끝에 자동으로 붙습니다.
  • 크기 조절 — 메모 모서리(대각선)를 드래그.
  • 새 메모 / 색 / 삭제 — 헤더의 + · 팔레트 · 휴지통.
  • 항상 위 · 투명 · 끝 도킹 — 작업을 방해하지 않게 화면 가장자리에 머뭅니다.

🍎 macOS 설치 방법

맥용은 "소스" 버전이라 Node.js (LTS)가 필요합니다.

  • zip을 풀고 EdgePostit 폴더 안의 start-mac.command 를 더블클릭하세요.
  • 처음 한 번은 자동으로 구성요소를 설치합니다(몇 분). 이후엔 더블클릭하면 바로 실행됩니다.
  • "확인되지 않은 개발자" 경고가 나오면 파일 우클릭 → 열기 로 한 번 허용하세요.

메모 저장 위치: ~/Library/Application Support/edge-postit/notes.json

🪟 Windows 설치 방법

  • zip을 받아 압축을 풀고 안의 실행 파일을 더블클릭하세요.
  • "Windows의 PC 보호" 경고가 나오면 추가 정보 → 실행 을 눌러주세요.
