# 엣지 포스트잇 (Edge Post-it)

화면 끝에 색 탭으로 숨어 있다가 마우스를 올리면 펼쳐지는 포스트잇 메모장(마크다운 지원).

## 다운로드

다운로드 페이지: **https://leedalnim.github.io/edge-post-it/**

- **macOS**: [`EdgePostit-mac.zip`](EdgePostit-mac.zip) — 소스 버전 (Node.js 필요)
- **Windows**: [최신 릴리스](https://github.com/leedalnim/edge-post-it/releases/latest)의 `EdgePostit-Windows.zip`

## 구성

- `index.html` — 다운로드 웹페이지 (GitHub Pages)
- `EdgePostit-mac.zip` — 맥용 소스 빌드

## Windows 빌드 올리는 법

용량이 큰 Windows 빌드는 일반 파일 업로드(25MB/100MB 제한) 대신
**GitHub Releases**(파일당 최대 2GB)에 올립니다. 자세한 방법은 저장소 관리자 안내를 참고하세요.
